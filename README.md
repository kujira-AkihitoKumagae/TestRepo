# ぶどう屋 - ブドウ販売ECサイト

🍇 国産・輸入ブドウの品種別EC販売サイト

## 概要

モバイルファーストのレスポンシブ対応で構築されたブドウ販売ECサイトです。

## 機能

- **商品一覧・詳細**: 品種、産地、糖度、サイズで絞り込み可能
- **カート機能**: 商品の追加・数量変更・削除
- **レスポンシブUI**: モバイル、タブレット、デスクトップ対応
- **検索・絞り込み**: ファセット検索対応

## 技術スタック

- **フロントエンド**: Next.js 16 + React 19 + TypeScript
- **スタイリング**: Tailwind CSS 4
- **画像最適化**: next/image (WebP/AVIF対応)

## ブレークポイント

| 画面サイズ | レイアウト |
|-----------|-----------|
| 0–768px | 単一カラム、モバイルメニュー |
| 768–1024px | 2カラム |
| 1024px以上 | 3〜4カラムグリッド |

## カラーパレット

- **深紫** (#3B1A52〜#8243A9): 熟度・高級感
- **赤・黄** (#DC2626, #F59E0B): 食欲喚起のアクセント
- **緑** (#16A34A): 新鮮さの表現

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# Lint
npm run lint
```

## ディレクトリ構成

```
src/
├── app/                    # App Router ページ
│   ├── page.tsx           # トップページ
│   ├── products/          # 商品一覧・詳細
│   └── cart/              # カートページ
├── components/            # UIコンポーネント
│   ├── layout/            # Header, Footer
│   ├── product/           # ProductCard, ProductFilter
│   └── ui/                # 共通UI
├── data/                  # サンプルデータ
└── types/                 # TypeScript型定義
```
